Dice Rolling Game (C++)
Description

A simple C++ console application that simulates rolling two dice. The program generates random values, displays each die result along with the total, and allows the user to roll again.

Features

Rolls two dice with values from 1 to 6

Displays individual dice values and total

Allows repeated rolls based on user input

Technologies Used

C++

Standard Libraries: <iostream>, <cstdlib>, <ctime>

How to Run
g++ dice_game.cpp -o dice_game
./dice_game

Sample Output
You rolled:
Die 1: 3
Die 2: 5
Total: 8

Roll again? : y
